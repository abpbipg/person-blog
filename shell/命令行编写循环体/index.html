    命令行编写循环体

    有时不想在脚本里面编写命令，需要直接在命令行编写可包含多条命令的语句，可以将每条语句用;隔开，就可以达到效果啦！

    $ i=0
$ while true; do echo $i; sleep 1; i=$(expr $i + 1); done
